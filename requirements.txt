# Hriday Chandra Portfolio Website - Requirements

## System Requirements
- Node.js >= 18.0.0
- npm >= 8.0.0 or yarn >= 1.22.0
- Modern web browser (Chrome, Firefox, Safari, Edge)

## Core Dependencies
react@^18.3.1                    # React library for building user interfaces
react-dom@^18.3.1                # React DOM rendering
typescript@^5.5.3                # TypeScript for type safety
vite@^5.4.2                      # Fast build tool and dev server

## UI & Styling
tailwindcss@^3.4.1              # Utility-first CSS framework
autoprefixer@^10.4.18           # CSS vendor prefixing
postcss@^8.4.35                 # CSS post-processing tool

## Animation & Interactions
framer-motion@^12.23.6          # Animation library for React
lucide-react@^0.344.0           # Beautiful icon library

## Development Tools
@vitejs/plugin-react@^4.3.1     # Vite React plugin
eslint@^9.9.1                   # JavaScript/TypeScript linting
@eslint/js@^9.9.1               # ESLint JavaScript rules
typescript-eslint@^8.3.0        # TypeScript ESLint rules
eslint-plugin-react-hooks@^5.1.0-rc.0  # React Hooks ESLint rules
eslint-plugin-react-refresh@^0.4.11    # React Refresh ESLint rules
globals@^15.9.0                 # Global variables for ESLint

## Type Definitions
@types/react@^18.3.5            # TypeScript definitions for React
@types/react-dom@^18.3.0        # TypeScript definitions for React DOM

## Installation Instructions
1. Clone the repository
2. Run `npm install` to install all dependencies
3. Run `npm run dev` to start the development server
4. Open http://localhost:5173 in your browser

## Build Instructions
1. Run `npm run build` to create production build
2. Run `npm run preview` to preview production build locally
3. Deploy the `dist` folder to your hosting provider

## Browser Support
